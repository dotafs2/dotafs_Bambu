# Hardware Projects

This directory is for external hardware and 3D-printing projects that are useful alongside this Bambu workspace.

Suggested layout:

- `vr-gloves/<project>/` for VR glove and hand-tracking hardware.
- `robotics/<project>/` for robot, actuator, and sensor hardware.
- `printer-mods/<project>/` for 3D-printer upgrades or accessories.
- `tools/<project>/` for jigs, fixtures, and workshop tools.

Each imported project should keep its original license and include an upstream snapshot note with the source URL, branch, and commit.
